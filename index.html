Number recognition using Turi Create

Step by step tutorial

About

Nov. 12th 2014

In this tutorial, we will learn how to use the TuriCreate API offered by Apple to create a number recognition app of our own using a pre-existing dataset.

We will be training our own machine learning model using this dataset, and exporting it for use in IOS devices.

This tutorial is designed with begginers in mind, who are interested in learning how to use the API for image recognition purposes.

Installation

Nov. 12th 2014

For this project, we would be needing:
1. A computer running mac OS.
2. Any version of python >= 2.7.14 which can be found here
3. A version of Xcode version >= enter number
4. An installation of PIP.
PIP is a tool for managing and installing python packages. It can be installed by opening the terminal and typing:

sudo easy_install pip

Lets go ahead and install TuriCreate by typing the following command in your terminal:

pip install turicreate

We are done with the installations for now, so let's go ahead and get started
